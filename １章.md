### レベル1： `print`文を使いこなそう

**問題**

`print()`文を複数回使って、画面に2行で自己紹介を表示するコードを書いてください。1行目に「こんにちは！私の名前は（あなたの名前）です。」、2行目に「プログラミングを学び始めました！」と表示されるようにしてください。


<details><summary>解答例</summary>

```python
print("こんにちは！私の名前は山田太郎です。")
print("プログラミングを学び始めました！")
```

**解説**
`print()`文は、実行されるたびに新しい行に出力します。2つの異なる文字列を別々の`print()`文で書くことで、2行にわたるメッセージを表示できます。

### レベル2： データ型の違いを理解しよう

**問題**

以下の3つのコードを実行すると、それぞれ異なる結果が表示されます。どのような結果になるか予想してください。

1.  `print(10 + 20)`
2.  `print("10" + "20")`
3.  `print("10" * 3)`



<details><summary>解答例</summary>

**解答**

**実行結果**

1.  `30`
2.  `1020`
3.  `101010`

**解説**

1.  `10` と `20` は **`int`型 (整数)** です。`int`型同士の `+` は、数学と同じ **足し算** として計算されるため、`30` と表示されます。
2.  `"10"` と `"20"` は **`str`型 (文字列)** です。`str`型同士の `+` は、文字を **連結** する処理になるため、`10` と `20` がつながった `1020` という文字列が表示されます。
3.  `"10"` は **`str`型 (文字列)** で、 `3` は **`int`型 (整数)** です。`str`型と`int`型の `*` (掛け算) は、文字列をその回数だけ **繰り返す** 処理になります。そのため、`"10"` が3回繰り返された `101010` という文字列が表示されます。

</details>

### レベル3： 変数を使って計算しよう

**問題**

あなたは八百屋で買い物をしています。変数を使って、リンゴとミカンの合計金額を計算してください。

*   `apple` という変数に `150` を代入します（リンゴの値段）。
*   `orange` という変数に `80` を代入します（ミカンの値段）。
*   `total_price` という新しい変数に、`apple` と `orange` を足した結果を代入します。
*   最後に `total_price` を `print()` してください。



<details><summary>解答例</summary>

```python
apple = 150
orange = 80
total_price = apple + orange
print(total_price)
```

**実行結果**

```
230
```

**解説**

`150 + 80` のように直接数値を計算するのではなく、一度変数という「箱」に入れることで、その値が「リンゴの値段」や「ミカンの値段」であることが分かりやすくなります。このように変数を使うことで、コードが何をしようとしているのかが明確になります。


</details>

### レベル4： 変数が上書きされる仕組みを考えよう

**問題**

以下のコードを実行すると、最終的に何が表示されますか？なぜその結果になるのか、変数 `point` の値が1行ずつどのように変化していくかを追いかけて説明してください。

```python
point = 100
point = point + 50
point = point - 20
print(point)
```


<details><summary>解答例</summary>

**実行結果**

```
130
```

**解説**

プログラミングにおける `=` は「代入」を意味し、右側の計算結果を左側の変数に **上書き保存** します。

1.  `point = 100`  
    `point` という箱に `100` が入ります。
2.  `point = point + 50`  
    まず右側の `point + 50` が計算されます。この時点での `point` は `100` なので、`100 + 50` で `150` となります。その結果が `point` に **上書き** され、`point` の中身は `150` になります。
3.  `point = point - 20`  
    同様に右側が計算されます。この時点での `point` は `150` なので、`150 - 20` で `130` となります。その結果が `point` に **上書き** され、`point` の中身は `130` になります。
4.  `print(point)`  
    最終的に `point` に入っている `130` が表示されます。

</details>

### レベル5： 値の入れ替えを応用しよう

**問題**

動画で習った「2つの変数の値を入れ替える」方法を応用して、**3つの変数の値を巡回させる** コードを `# add_here!` の部分に書いてください。

具体的には、`a` の値を `b` に、`b` の値を `c` に、`c` の値を `a` に入れ替えてください。

**条件**
*   新しく値を生成しないでください（例: `a = 30` のように直接書かないでください）。
*   演算子 (`+, -, *, /` など) は使わないでください。
*   既存のコード（`===`で囲まれた部分）は変更しないでください。

```python
# === 既存のコード (変更しない) ===
a = 10
b = 20
c = 30
# ===============================

# add_here!
# ここにコードを追加して値を巡回させる (a→b, b→c, c→a)


# === 既存のコード (変更しない) ===
print(a) # 期待する出力: 30
print(b) # 期待する出力: 10
print(c) # 期待する出力: 20
# ===============================
```

<details><summary>解答例</summary>

**解答**

```python
# === 既存のコード (変更しない) ===
a = 10
b = 20
c = 30
# ===============================

# add_here!
temp = a  # まずaの値を一時的にtempに退避させる (tempは10になる)
a = c     # aにcの値を入れる (aは30になる)
c = b     # cにbの値を入れる (cは20になる)
b = temp  # bに退避しておいた元のaの値(temp)を入れる (bは10になる)


# === 既存のコード (変更しない) ===
print(a)
print(b)
print(c)
# ===============================
```

**解説**

2つの値を入れ替えるときと同様に、一度どこかに値を退避させないと、元の値が上書きされて消えてしまいます。

1.  まず、`a` の値 `10` が上書きされる前に、新しい変数 `temp` に退避させます。
2.  これで `a` の箱は空にしても大丈夫なので、`a` に `c` の値 `30` を入れます。
3.  次に、`c` に `b` の値 `20` を入れます。
4.  最後に、`b` に `temp` に退避しておいた元の `a` の値 `10` を入れます。

このように、一時的な変数（この場合は `temp`）をうまく使うことで、複数の変数の値を正しく入れ替えることができます。

</details>
