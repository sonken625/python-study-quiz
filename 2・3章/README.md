### レベル1：基本的な演算

**問題**

りんごが8個、みかんが5個あります。
1.  りんごとみかんを合わせた合計の個数を計算して表示してください。
2.  りんごはみかんより何個多いか計算して表示してください。


<details><summary>解答例</summary>

```python
# 1. 合計の個数
print(8 + 5)

# 2. 個数の差
print(8 - 5)
```

**実行結果**

```
13
3
```

**解説**
動画で学んだ最も基本的な算術演算子 `+` と `-` を使って計算し、`print()` で結果を表示します。

</details>

### レベル2：変数と値の更新

**問題**

ゲームのポイントを管理する変数 `point` を作り、最初に `100` を代入します。
その後、以下のイベントが発生したとします。

1.  ボーナスで50ポイント加算された。
2.  アイテム購入で30ポイント消費した。

最終的に `point` がいくつになるか計算し、その値を表示するプログラムを書いてください。
**注意点：** `point` という1つの変数だけを更新しながら使ってください。


<details><summary>解答例</summary>

```python
point = 100
point = point + 50
point = point - 30
print(point)
```

**実行結果**

```
120
```

**解説**
変数に値を代入し、その変数自身を使って計算結果を「再代入（上書き保存）」する考え方が重要です。`point = point + 50` のように、右辺の計算結果を左辺の変数に改めて入れ直すことで、値を更新していきます。

</details>

### レベル3：簡単な条件分岐 (if-else)

**問題**

あるアトラクションには「身長120cm以上」という利用条件があります。
変数 `height` にあなたの身長（cm、整数）を代入し、もし身長が120cm以上なら「アトラクションをご利用いただけます」、そうでなければ「申し訳ございませんが、ご利用いただけません」と表示するプログラムを書いてください。


<details><summary>解答例</summary>

```python
height = 130 # ここに自分の身長を入れる

if height >= 120:
    print("アトラクションをご利用いただけます")
else:
    print("申し訳ございませんが、ご利用いただけません")
```

**実行結果（height = 130の場合）**

```
アトラクションをご利用いただけます
```

**実行結果（height = 110の場合）**

```
申し訳ございませんが、ご利用いただけません
```

**解説**
`>=` という比較演算子を使って条件式を作り、`if`文で条件分岐させます。条件が正しい場合 (`True`) とそうでない場合 (`False`) で処理を分ける `if-else` 構文の応用です。

</details>

### レベル3：複数の条件分岐 (if-elif-else) と論理演算子

**問題**

ある映画館の入場料金は、年齢によって変わります。
- 12歳以下は「子供料金：500円」
- 13歳以上、18歳以下は「学生料金：1000円」
- 19歳以上は「大人料金：1500円」

変数 `age` に年齢を代入し、その年齢に応じた料金案内を表示するプログラムを書いてください。


<details><summary>解答例</summary>

```python
age = 15 # ここに年齢を入れる

if age >= 19:
    print("大人料金：1500円")
elif age >= 13:
    print("学生料金：1000円")
else:
    print("子供料金：500円")
```
**実行結果（age = 15の場合）**

```
学生料金：1000円
```

**解説**
この問題のポイントは、`if`と`elif`の評価順序です。プログラムは上から順に条件を評価します。
`age`が15の場合、
1. `if age >= 19:` は `False` なので、次の `elif` に進みます。
2. `elif age >= 13:` は `True` なので、`"学生料金：1000円"` を表示し、この `if`文全体の処理を終了します。
`age >= 13 and age <= 18` のように `and` を使って書くこともできますが、`if-elif` の順序をうまく使うことで、よりシンプルに記述できます。

</details>

### レベル3.5（新しい演算子 `//`）

**問題**
`100`個のアメがあります。このアメを`7`人の子供に、できるだけ平等に分けます。
1.  子供1人あたり何個のアメをもらえるか？
2.  最終的に何個のアメが余るか？

この2つを計算して表示するプログラムを書いてください。

**新しい演算子の紹介：**
割り算には、小数点以下を切り捨てる `//` という演算子（**整数除算**）があります。
例えば、`10 / 3` は `3.333...` になりますが、`10 // 3` は `3` になります。

<details><summary>解答例</summary>

**答え**
```python
candies = 100
children = 7

# 1. 1人あたりの個数 (小数点以下は切り捨て)
per_child = candies // children

# 2. 余った個数
remainder = candies % children

print("1人あたり " + str(per_child) + " 個です")
print("余りは " + str(remainder) + " 個です")
```
**実行結果**
```
1人あたり 14 個です
余りは 2 個です
```

**解説**
`//`（整数除算）は商を、`%`（剰余）は余りを求めるのに使えます。この2つはセットでよく使われるので覚えておくと便利です。この問題を通じて、`//` の使い方を具体的に学ぶことができます。

</details>

### レベル3.5（新しい演算子 `**`）

**問題**
BMI（ボディマス指数）は、体重(kg) ÷ (身長(m) × 身長(m)) で計算されます。
変数 `weight_kg` に体重(kg)を、`height_m` に身長(m)を代入し、BMIを計算して表示するプログラムを書いてください。

**新しい演算子の紹介：**
`**` は **累乗** を計算する演算子です。
例えば、`3 ** 2` は「3の2乗」を意味し、結果は `9` になります。`2 ** 3` は「2の3乗」で `8` です。

```python
weight_kg = 65
height_m = 1.75

# ここにコードを書いてください
```


<details><summary>解答例</summary>

**答え**
```python
weight_kg = 65
height_m = 1.75

# 身長の2乗を ** 演算子で計算する
bmi = weight_kg / (height_m ** 2)

print("あなたのBMIは " + str(bmi) + " です")
```
**実行結果**
```
あなたのBMIは 21.224489795918366 です
```

**解説**
「身長 × 身長」を `height_m ** 2` と書くことで、コードがより簡潔になります。`**` 演算子を使うことで、累乗の計算が簡単に行えることを学びます。実用的な計算式の中で新しい演算子に触れることで、記憶に残りやすくなります。

</details>


### レベル4: うるう年判定プログラム

**問題**

西暦年を格納する変数 `year` を用意し、その年が「うるう年」か「平年（うるう年ではない年）」かを判定して表示するプログラムを作成してください。

**【うるう年のルール】**
1.  西暦年が4で割り切れる年は、うるう年である。
2.  ただし、1.の条件を満たしていても、100で割り切れる年は、平年である。
3.  ただし、2.の条件を満たしていても、400で割り切れる年は、うるう年である。

**ヒント：** 剰余演算子 `%` と、論理演算子 `and` `or` を組み合わせることで、複雑な条件を1つの `if` 文で表現できます。
<details><summary>解答例</summary>

**答え**

```python
year = 2024 # ここに西暦年を入れる

# 条件式を()で囲むと見やすくなります
if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print(str(year) + "年は、うるう年です")
else:
    print(str(year) + "年は、平年です")
```

**実行結果（year = 2024の場合）**

```
2024年は、うるう年です
```

**実行結果（year = 2100の場合）**
```
2100年は、平年です
```

**解説**
この問題は、ルールを正確にプログラムの条件式に落とし込む論理的思考が求められます。
`(year % 4 == 0 and year % 100 != 0)` の部分が「4で割り切れる **かつ** 100で割り切れない年」というルール1と2を表しています。
それに `or (year % 400 == 0)` を組み合わせることで、「 **または** 400で割り切れる年」というルール3を追加しています。このように `and` や `or` を使うことで、複雑な条件を組み立てることができます。

</details>

### レベル5：剰余演算子と論理演算子の組み合わせ

**問題**

あなたはパーティーの幹事です。参加者の人数（変数 `num_of_people`）と、テーブル1つあたりの座席数（変数 `seats_per_table`）が決まっています。
全ての参加者が座るためには、テーブルが最低何台必要か計算し、表示するプログラムを書いてください。
ただし、席が余ってもテーブルを分けることはできません。


```python
num_of_people = 23
seats_per_table = 5

# ここにコードを書いてください
```

**ヒント：** 割り算の「商」と「余り」をうまく使う必要があります。余りが出た場合はどうすればよいでしょうか？


<details><summary>解答例</summary>

```python
num_of_people = 23
seats_per_table = 5

# 割り切れる場合のテーブル数
tables = num_of_people // seats_per_table # // は割り算の商（整数）を求める演算子ですが、/でも構いません

# 席が余るかどうかを判定
if num_of_people % seats_per_table != 0:
    tables = tables + 1

print(tables)
```
**実行結果（num_of_people = 23, seats_per_table = 5 の場合）**

```
5
```

**解説**
これは論理的思考を必要とする問題です。
1.  まず、単純に人数を座席数で割り、必要なテーブル数の基本（商）を求めます。(`23 // 5` は `4`)
2.  次に、剰余演算子 `%` を使って、席が余る（＝割り切れない）かどうかを判定します。`23 % 5` の結果は `3` なので、`!= 0` (0ではない) という条件は `True` になります。
3.  条件が `True` の場合、基本のテーブル数に加えて、余った人たちのためのテーブルがもう1台必要になるため、`tables` に `1` を足します。
4.  結果として `4 + 1` で `5` が表示されます。

もし参加者が20人できっかり割り切れる場合、`20 % 5` は `0` になるため `if`文の条件は `False` となり、`tables` は `4` のまま正しく計算されます。

</details>

#### レベル5：到着時刻計算プログラム

**問題**

出発時刻（`start_hour`）と、移動にかかる時間（`duration_hour`）が与えられています。24時間表記で、到着時刻が何時になるか計算して表示するプログラムを作成してください。

```python
start_hour = 23
duration_hour = 4

# ここにコードを書いてください
```


**ヒント：** 例えば、23時に出発して2時間後、時刻は25時ではなく1時になります。剰余演算子 `%` は、このような周期的な計算（一周して0に戻る計算）に非常に便利です。

<details><summary>解答例</summary>

**答え**
```python
start_hour = 23
duration_hour = 4

# (出発時刻 + 移動時間) を24で割った余りが到着時刻になる
arrival_hour = (start_hour + duration_hour) % 24

print("到着時刻は" + str(arrival_hour) + "時です")
```

**実行結果（start_hour = 23, duration_hour = 4の場合）**

```
到着時刻は3時です
```

**解説**
この問題の鍵は、時刻が24になると0に戻るという周期性に気づき、それを剰余演算子 `%` で解決することです。
`23 + 4` は `27` ですが、`27 % 24` を計算すると余りは `3` となり、これがそのまま24時間表記の時刻になります。`if`文を使わなくても、演算子だけでスマートに解くことができる良い例です。

</details>
